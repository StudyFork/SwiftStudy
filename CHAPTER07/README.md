# CHAPTER 07 함수

함수가 도대체 뭔데?!
* 함수란 프로그램 소스 코드에서 **일정한 동작을 수행** 하는 코드 집합체 이다.
* Swift에서의 함수는 **1급 객체(first class citizen)** 이기 때문에 하나의 값으로도 사용할 수 있습니다.

다음 조건을 모두 충족하는 객체를 **1급 객체(first class citizen)** 라고 한다
* 변수에 담을 수 있다.
* 파라미터로 전달할 수 있다.
* 변환값으로 사용할 수 있다.
